Data Overview

This is a brief overview of the data that we used to feed the application and how it was applied. The idea was to be as transparent as possible since we sourced this data from various places across the web and then manually geocoded it to suit the needs of the application. Below is a list of the resources that we used during the information gathering process. Because of the great diversity in single malt scotch whiskies, it was important to get information from as many different sources as possible.

Data Sources

Name Information Gathered
Scotch Whisky Descriptions, regions, facts, and production capacity
Whisky Base Founding dates, regions, still counts, locations, and production capacity
Malt Madness Founding dates, regions, and locations
Reddit Scotch Community Tasting notes, region information, introductory malt information
Flaviar Scotch Blog Tasting notes, region information
Scotch Whisky Association Historical information, tasting notes, general scotch facts
